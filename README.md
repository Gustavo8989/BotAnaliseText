Função analise de textual formulando um resumo
------------------------------------------------
Por enquanto, vou focar apenas em arquivos PDF para essa análise. Adiante, irei inserir a capacidade de análise de imagens.

Visão computacional:
    1. Analisar as imagens e extrair o texto

Processamento de Linguagem Natural(PLN):
    1. Capacidade de interpretação
    2. Organizar as informações 
    3. Criar um resumo 


A princípio, o sistema vai funcionar apenas com arquivos pré-inicializados, ou seja, o arquivo terá que ser passado antes do sistema rodar. De acordo com o andamento do projeto, será necessário colocá-lo em um site, possivelmente por meio de uma API.
